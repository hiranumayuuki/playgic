# YouTube — Rebel Sheep（自由とレバレッジ）

## チャンネル戦略

| 項目 | 内容 |
|---|---|
| チャンネル名 | Rebel Sheep（自由とレバレッジ） |
| ターゲット | 会社員×副業×投資でFIREを目指す人 |
| フォーマット | 顔なし × 声あり・短中尺（3〜8分）（2026-05-26 確定） |
| 編集ツール | CapCut（無料版）※透かしが出る場合はiMovieでフィニッシュ |
| 差別化 | リアルな数字・リアルな軌跡。綺麗事なし、等身大。 |

## フェーズ設計

```
今（顔なし・匿名）
  → 本業を辞めるタイミングで顔出し解禁・フルコミット
  → 「辞めました」動画がバズる設計
```

---

## 画面構成（確定）

**スクリーンレコーディング＋ナレーション**

- 証券口座・チャート・計算シートの実画面を見せながら声で解説
- モザイク必須：口座番号・氏名・ログインID
- 資産額は丸めてOK（例：「約1,800万」）
- 数字テロップ必須：「2.15%」「月896円」「RSI値」など
- BGM：Lo-fi系、主張しすぎないもの

---

## 収録フロー

```
① ナレーション録音（スクリプトを見ながら音声のみ収録）
② 音声を聞きながらスクリーン収録
③ CapCutで合体・編集・テロップ・モザイク処理
④ サムネイル作成（Canva等）
⑤ YouTube投稿設定（タイトル・説明文・タグ）
```

## 所要時間（1本目の目安）

| 工程 | 時間 |
|---|---|
| 収録（ナレ＋画面） | 約1〜1.5時間 |
| 編集・テロップ・BGM・モザイク | 約2〜3時間 |
| サムネイル＋投稿設定 | 約1時間 |
| **合計** | **約4〜5時間** |

**スケジュール案：** 今週平日夜2日で収録 → 来週編集・投稿
